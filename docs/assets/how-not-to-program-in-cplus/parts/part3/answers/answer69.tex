مشکل این است که برنامه‌نویس نوشته است:
\begin{LTR}
    %@formatter:off
        \begin{lstlisting}[style=C++Style]
            std::cout << "Hello World!/n";
        \end{lstlisting}
    %@formatter:on
\end{LTR}

به جای این که بنویسد:
\begin{LTR}
    %@formatter:off
        \begin{lstlisting}[style=C++Style]
            std::cout << "Hello World!\n";
        \end{lstlisting}
    %@formatter:on
\end{LTR}

بنابراین خروجی به صورت لفظ‌به‌لفظه برابر است با:
\LTR\noindent
\lr{\texttt{Hello World/n}}
\RTL
