\paragraph{}\label{answer:47}
عبارت \lr{\texttt{if (amount = 0)}} مقدار \lr{\texttt{0}} را به \lr{\texttt{amount}} می‌دهد، سپس نتیجه را مقایسه می‌کند تا ببیند که برابر صفر نیست. برابر صفر است، لذا عبارت \lr{\texttt{else}} اجرا می‌شود. برنامه‌نویس باید می‌نوشت:
\LTR\noindent
\lr{\texttt{if (amount == 0)}}
\RTL
