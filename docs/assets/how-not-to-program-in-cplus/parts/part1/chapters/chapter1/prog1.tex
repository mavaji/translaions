\section[\lr{"Hello World"}]{\lr{Hello World} (راهنمایی \ref{hint:228}، جواب \ref{answer:6})}
\paragraph{}\label{prog:1}
به نظر می‌رسد که \lr{"Hello World"}، اولین برنامه در هر کتابِ برنامه‌نویسی باشد و فرقی نمی‌کند. ولی این یکی مشکل دارد. چگونه می‌توان چیزی به سادگیِ یک \lr{"Hello World"} را خراب کرد؟ نگاه کنید:

\begin{LTR}
    %@formatter:off
        \begin{lstlisting}[style=C++Style]
         /************************************************
         * The "standard" hello world program. *
         *************************************************/
         #include <iostream>

         void main(void)
         {
            std::cout << “Hello world!\n”;
         }
        \end{lstlisting}
        %@formatter:on
\end{LTR}
