\paragraph{}\label{answer:42}
یک عدد تک‌بیتیِ علامت‌دار می‌تواند دو مقدار داشته باشد: \lr{\texttt{0}} و \lr{\texttt{-1}}. عبارت:
\lr{\texttt{printer\_status.online = 1;}}
درست کار نمی‌کند چون یک تک‌بیت نمی‌تواند مقدار \lr{\texttt{1}} داشته باشد. (لذا سرریز می‌کند و مقدار \lr{\texttt{-1}} را به متغیر می‌دهد!) نتیجه این است که عبارت بعدی \lr{\texttt{if (printer\_status == 1)}} نیز درست کار نمی‌کند. تک‌بیت‌ها باید \lr{\texttt{unsigned}} باشند.
